# DMAIC Case Study — Reducing Supplier Non-Compliance in Procurement

**Helen Lam** | MBA Candidate, Supply Chain Analytics | Saint Leo University

## Project Summary
Applied DMAIC methodology to 777 real procurement purchase orders 
across 5 suppliers (2022–2024). Dataset: Kaggle CC0 Public Domain.

| Metric | Before | Target |
|--------|--------|--------|
| NC Rate | 17.6% | 8.0% |
| Sigma Level | 2.43 | 3.1+ |
| Contract Value at Risk | $6,986,051 | $3,175,478 |
| Projected Recovery | — | $3,810,573/yr |

## What's in this repo
- `DMAIC_CaseStudy_HelenLam.pdf` — Full case study with MBB Lens + Apex analysis
- `DMAIC_CaseStudy_HelenLam.pptx` — Portfolio deck

## Key findings
- CTQ selection: p = 0.7612 (wrong) vs p = 0.0000000000 (correct) — same dataset
- Root cause: No dynamic contract review mechanism at Delta_Logistics (48.9% of all violations)
- Apex: Goodhart's Law applied to own control plan — 3 vulnerabilities identified and counter-measured
